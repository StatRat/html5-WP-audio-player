HTML5 WP Audio Player

This project is a version of the awesome WordPress Audio Player by Martin Laine for browsers that support HTML5’s <audio> tag, with fallback to the original WP Audio Player for browsers that don’t.

Audio sample is Crookshanks by Palace Cat, via duplokat

See the standalone WordPress Audio Player page for information on using and configuring the Flash player.
